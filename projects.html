AudiOrigins
React app that takes user input of an artist and queries theaudiodb.com to GET their hometown and other data, .then() queries mapbox's geocoding API for the coordinates of that location and zooms to it on the embedded mapbox instance. Uses best practices for state management.

Schedeo
Event planning MVC app, key feature is the ability of users to vote on the date of an event. Handlebars templates delivered by Node/Express server. Authentication with Passport.js, image hosting with Cloudinary. Persistent data stored in MongoDB, uses complex, completely custom models defined with mongoose. Bootstrap used for responsive, mobile-first styling. Built with one other colleague using GitHub. You may use the following credentials if you don't want to make an account:

Reimagining the USA
A QGIS project of mine from a few years ago that uses spatial analysis to divide the states up in a more logical way. Exported to and overlayed on an OpenLayers webmap instance. Uses jQuery for interactivity - assigning event listeners and DOM manipulation. Custom CSS.

Best practices used to keep it DRY and SOLID. Not mobile-friendly.
