Borrowly
Peer-to-peer app for borrowing/lending things, built with MERN stack. Single-page app uses react-router-dom for rendering views and axios for sending requests. Express backend serves static files and configures REST API endpoints that respond with JSON retrieved from MongoDB. Google maps APIs (with no helper packages) used for map instances and geocoding. Best practices used for component lifecycles, managing state, and graceful error handling. Search "Portland" on the homepage to interact with existing data.
